# 🎓 Exemplos de Fundamentos do React Native

Este projeto contém exemplos didáticos para ensinar os fundamentos do React aplicados ao desenvolvimento mobile com Expo.

## 📚 Temas Abordados

### 1. **JSX** (`exemplo-1-jsx.tsx`)

- Sintaxe JSX básica
- Interpolação de variáveis com `{}`
- Expressões JavaScript dentro do JSX
- Operadores ternários

**Conceitos demonstrados:**

- Como usar variáveis dentro do JSX
- Realizar cálculos e operações
- Renderização condicional simples

---

### 2. **Componentes Funcionais** (`exemplo-2-componentes.tsx`)

- Criação de componentes funcionais
- Composição de componentes
- Reutilização de código

**Conceitos demonstrados:**

- Componentes como funções que retornam JSX
- Separação de responsabilidades
- Organização modular do código

---

### 3. **Props (Propriedades)** (`exemplo-3-props.tsx`)

- Passagem de dados entre componentes
- Tipagem de Props com TypeScript
- Props opcionais com valores padrão
- Desestruturação de Props

**Conceitos demonstrados:**

- Como passar dados de pai para filho
- Tipagem forte com TypeScript
- Props opcionais usando `?`
- Valores padrão para Props

---

### 4. **Exemplo Completo** (`exemplo-4-completo.tsx`)

- Aplicação prática combinando todos os conceitos
- Lista de tarefas com diferentes estados
- Cálculo de estatísticas
- Renderização de listas com `map()`

**Conceitos demonstrados:**

- Integração de JSX + Componentes + Props
- Manipulação de arrays
- Estilos condicionais
- Aplicação real e funcional

## 🎯 Objetivos de Aprendizado

Ao final desta aula, os alunos devem ser capazes de:

- ✅ Entender e escrever JSX
- ✅ Criar componentes funcionais reutilizáveis
- ✅ Passar dados entre componentes usando Props
- ✅ Tipar componentes com TypeScript
- ✅ Combinar todos os conceitos em uma aplicação funcional
